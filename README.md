API Testing Project

Projeto desenvolvido durante o Bootcamp de QA da TripleTen.

Objetivo

Validar endpoints de API por meio de cenários positivos e negativos, verificando códigos de resposta, comportamento do sistema e tratamento de erros.

Atividades Realizadas

* Criação de casos de teste
* Testes positivos e negativos
* Validação de respostas da API
* Registro de defeitos encontrados

Ferramentas Utilizadas

* Postman
* Google Sheets
* Jira

Resultado

Execução dos cenários de teste e documentação dos resultados obtidos durante a validação da API.
